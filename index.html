Welcome to Dummy Ruby.

Dummy Ruby is written to replace lorem ipsum dummy text generator. It can currently produce paragraphs and lists. It does it by using random passages from its library of public domain books. It is stable enough for release. But it is also constantly under development.

Documentation

Documentation for the generator is available at https://github.com/adhithyan15/dummyruby/wiki/Documentation.
